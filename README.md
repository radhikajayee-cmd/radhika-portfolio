## 🚀 Getting Started

First, clone the repository:

```bash
git clone https://github.com/your-username/radhikajayee-portfolio.git
cd radhikajayee-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open browser:

```bash
http://localhost:3000
```
